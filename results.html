Microsoft 365 results connection

The games are ready to send every finished result to one Power Automate flow. The flow should receive the result, then add a row to the GameResults table in the Excel workbook.

1. Excel Create or upload the workbook named Mr Barron Game Results.xlsx. It must contain a table called GameResults.
2. Power Automate Create a cloud flow with the trigger When an HTTP request is received, then add a row into the Excel table.
3. Website Paste the flow's HTTP URL into results-config.js as CLASSROOM_RESULTS_ENDPOINT.
Columns Timestamp, Game, PlayerName, ClassLevel, Score, Correct, Attempts, Accuracy, BestStreak, Level, topic scores, topic correct counts, DurationSeconds, SourceUrl.
